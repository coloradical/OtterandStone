Services

Otter & Stone can help support any size group, meeting virtually or in hybrid format. For example: conferences, institutes, symposia, forums, panels, team-building events, and social events. We provide the following offerings:

Event Prep

  • Agenda Development
  • Facilitation Services
  • Creative Ideas for Engagement
  • Graphic Support / Event Branding
  • Event Homepage and Registration Page Creation
  • Survey and Evaluation Ideation and Implementation
  • Copywriting Content Sent to Participants and Presenters
  • Technical Support Before the Event for Presenters

Event Assistance

  • Running the technical back end of an Event
  • Setting up the Event Link and Settings
  • Welcoming Participants into the Events
  • Reviewing Basic Application Tools with Participants
  • Managing Breakout Rooms
  • Scren Sharing, Audio Sharing and Other Application Support for Presenters
  • Ensuring Facilitators are Uninterrupted by Participants
  • Recording Attendance
  • Documenting the Event with Photos
  • Recording, Editing and Posting Event Videos
